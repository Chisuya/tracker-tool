About Time Tool

Time Tool is a desktop time tracking application that automatically monitors your active applications and syncs your tracked time to Google Calendar with colour-coded events. Perfect for freelancers, students, and anyone who wants to understand how they spend their time.

How It Works

  1. Download and extract Time Tool
  2. Run the application
  3. Authorize Google Calendar access (one-time setup)
  4. Start tracking automatically!

All your data stays on your computer. Time Tool only accesses your Google Calendar to create time-tracking events - no other data is collected or sent anywhere.

System Requirements

  • Windows 10/11
  • 50 MB disk space
  • Internet connection (for Google Calendar sync)

Open Source

Time Tool is open source and built with Python, CustomTKinter, Google Calendar API, and SQLite. You can review the code, contribute, or report issues on GitHub.
